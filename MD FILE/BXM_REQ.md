## 프로젝트 개요

BXM팀 온라인 외부몰(네이버·오늘의집·CJ몰·SSG·쿠팡)의 월별 수주/매출 현황을 구글시트에서 읽어와 웹에서 조회하는 뷰어.
Google Sheets API(gspread + 서비스 계정)로 인증하여 데이터를 읽고, 기존 사업부 앱의 추가 페이지(pages/bxm.py)로 통합한다.

| 데이터 쓰기 | 없음 (읽기 전용 뷰어) |

## 채널 정의

채널 목록과 PM은 변경/추가될 수 있으므로 코드에 고정하지 않고 **엑셀 파일(BXM_채널설정.xlsx)** 에서 읽어온다.

| 컬럼 | 설명 |
|------|------|
| 채널명 | 시트에 표시되는 채널 이름 (예: 네이버, CJ몰) |
| PM | 담당 PM 이름 |
| 목표여부 | 목표 데이터 존재 여부 (Y/N) |
| 순서 | 테이블 표시 순서 |

> 현재 채널 (초기값, 엑셀에 입력 필요):
> 온라인합계 / 네이버(유지원·이유정, 목표O) / 오늘의집(김도경) / CJ몰(김은지) / SSG(김도경) / 쿠팡(유지원·이유정)
