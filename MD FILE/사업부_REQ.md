# 요구사항 정의 (사업부_REQ.md)
> 대상: DESKER 사업부 전체 월별마감 관리 시스템
> 최종 갱신: 2026-04-22 (사이드바 과거실적 페이지 분리)

---

## 프로젝트 개요
DESKER 사업부 영업팀이 매달 엑셀로 작성하던 수주/매출 월별마감 양식을 웹 기반 시스템으로 전환.
Google Sheets를 데이터 소스로 연동하고, 대시보드에서 실적·목표·YoY를 시각화한다.

---

## 채널 정의

| 구분 | 채널명 | 설명 |
|------|--------|------|
| 입력 채널 | 온라인외부몰 | BXM (외부 온라인 마켓) |
| 입력 채널 | 오프라인 | CXM 오프라인 |
| 입력 채널 | 공식몰+MATE | CXM 공식몰 |
| 자동 계산 | B2C 소계 | 온라인외부몰 + 오프라인 |
| 자동 계산 | 합계 | 전체 채널 합 |

## 팀/고객 구분

| 구분 방식 | 뷰 | 포함 채널 |
|---------|-----|---------|
| 팀 구분 | 전체 | 합계 |
| 팀 구분 | BXM | 온라인외부몰 |
| 팀 구분 | CXM | 오프라인 + 공식몰+MATE |
| 고객 구분 | 전체 | 합계 |
| 고객 구분 | B2C | 온라인외부몰 + 오프라인 |
| 고객 구분 | B2B | 공식몰+MATE |

## 기능 요구사항

### 1. 인증
- [x] 비밀번호 기반 로그인 (.streamlit/secrets.toml)
- [x] 세션 기반 인증 유지
- [x] 로그아웃

### 2. 데이터 소스
- [x] Google Sheets 연동 (sheets.py) — 실적·목표·EXTRA 데이터 가져오기
- [x] 로컬 SQLite DB (demo.db) — 가져온 데이터 캐시 저장
- [x] 최초 접속 시 연동 필수 게이트 (연동 없이 대시보드 접근 불가)
- [x] 사이드바 "시트연동 새로고침" 버튼으로 재연동

### 3. 대시보드

#### 3.1 필터
- [x] 연도 선택
- [x] 구분 선택 (수주/매출) — 구분방식 변경 시 수주로 초기화
- [x] 구분방식 (팀 구분 / 고객 구분)
- [x] 뷰 선택 (전체/BXM/CXM 또는 전체/B2C/B2B)

#### 3.2 KPI 카드 (6개)
- [x] 연간 목표
- [x] 실적 목표 (YTD 기간)
- [x] 누적 실적 (YTD)
- [x] 달성률 (YTD실적 / YTD목표)
- [x] YoY 실적 (전년 동기)
- [x] YoY 성장률

#### 3.3 월별 추이 차트
- [x] 전년/당해 실적 묶음 막대 차트
- [x] YoY 성장률 꺾은선 오버레이 (우축)
- [x] 실적 있는 월만 YoY 성장률 표시

#### 3.4 목표 대비 달성률 게이지
- [x] 연간 목표 대비 누적 실적 가로 바 차트
- [x] 분기 가이드라인 (Q1~Q4 점선 + 라벨)

#### 3.5 채널별 실적 비중 도넛 차트
- [x] 당해 YTD / 전년 동기 나란히 표시
- [x] 외부 레이블: 채널명, 금액(증감액), 비중%(비중변화%p)
- [x] 중앙: 합계 금액 + 증감액 + 성장률% (줄바꿈)
- [x] BXM·B2B 뷰는 도넛 미표시 → 요약 metric으로 대체
- [x] 고객 구분 + 전체 뷰: B2C / B2B 2분할 도넛

#### 3.6 월별 상세 테이블
- [x] 행: 목표/실적/전년실적/달성률/YoY 성장률
- [x] 열: 1월~12월 + 총합
- [x] 총합 열 강조 (연한 빨강 배경)
- [x] 달성률/YoY 색상 조건부 서식

#### 3.7 분기별 현황 (expander)
- [x] Q1~Q4 + 연간 열
- [x] 목표/실적/달성률/YoY 성장률 행

#### 3.8 채널별 누적 현황 테이블
- [x] 뷰에 따라 채널 행 구성
- [x] 연간목표/누적실적/기간달성률/연간달성률/YoY실적/YoY성장률
- [x] 채널별 분기 현황 (expander)
- [x] 고객 구분 + 전체 뷰: 사업부 전체 / B2C / B2B 계층 표시

#### 3.9 EXTRA 현황 테이블
- [x] 당해 / 전년 EXTRA 데이터 표시
- [x] 채널: 합계·B2C온라인·B2C오프라인·B2B(특판/직판)
- [x] 백만원 단위 ± 포맷

### 4. 설정 페이지
- [x] 연간 목표 설정 (채널별 월별 입력, 수정 가능) — 탭 없이 단일 섹션

  - [x] 채널별 연간 합계 실시간 표시

### 5. 과거 실적 페이지 (사이드바 메뉴 분리)

- [x] 과거 실적 입력 (최초 1회, 이후 읽기 전용) — 설정 탭에서 독립 페이지로 분리

  - [x] locked view: 연간합계 열(맨 앞), 합계 행, 채널명 정규화
